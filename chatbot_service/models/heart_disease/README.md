# Heart Disease Prediction Models

## Why we use these models
These files contain pre-trained machine learning pipelines (such as Stacking Models, Decision Trees, MLPs, Random Forests, XGBoost, etc.) that are used to predict the likelihood of heart disease in users based on various tabular health metrics. Rather than relying on a generalized LLM for statistical predictions, we utilize these highly specialized, trained tabular data models to process user health inputs and output reliable risk assessments. These risk assessments are then integrated into the system to guide the chatbot conversations and provide personalized, data-driven medical insights.
