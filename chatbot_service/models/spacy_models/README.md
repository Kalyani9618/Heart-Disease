# SpaCy Models

## Why we use this model
This folder contains the `en_core_web_trf` (Transformer-based) SpaCy model used for advanced Natural Language Processing. We utilize SpaCy in our data processing pipelines for critical text preprocessing, entity extraction, and syntactic analysis on user inputs or medical texts. Employing this robust transformer-based model allows the system to accurately identify medical terminology, parse complex sentence structures, and thoroughly structure the text representations before they are utilized by our prediction models, embedding models, and LLMs.
